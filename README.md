# 💻 Portafolio - Denis Salinas

Bienvenido a mi portafolio personal de desarrollo web.  
Este sitio presenta mi experiencia, habilidades, proyectos y formación como desarrollador.

🌐 Sitio web:  
https://HariiSeldonZLV.github.io

---

## 🚀 Tecnologías utilizadas

Este portafolio fue desarrollado utilizando:

- Vue 3
- Vue Router
- Vuetify
- TypeScript
- Vite
- HTML5
- CSS3

---

## 📂 Secciones del portafolio

El sitio está organizado en las siguientes secciones:

- **Historia**  
  Un recorrido por mi trayectoria personal y profesional.

- **Skills**  
  Tecnologías y herramientas que manejo.

- **Proyectos**  
  Aplicaciones y desarrollos realizados.

- **Experiencia**  
  Experiencia laboral y proyectos relevantes.

- **Estudios**  
  Certificaciones y formación académica.

- **Contacto**  
  Formas de contactarme profesionalmente.

---

## 🎨 Características del diseño

El portafolio tiene una estética inspirada en interfaces **retro digitales y cyberpunk**, incluyendo:

- Animaciones LED
- Efectos de iluminación
- Interfaz minimalista
- Navegación interactiva mediante tarjetas animadas

---

## 📦 Instalación del proyecto

Si deseas ejecutar el proyecto localmente:

```bash
git clone https://github.com/HariiSeldonZLV/HariiSeldonZLV.github.io.git
cd HariiSeldonZLV.github.io
npm install
npm run dev
```
